If
É uma condicional que executa uma afirmação caso a condição determinada seja verdadeira. Ao contrário, seguirá para o else if. Exemplo:
- if (intensCafé === 'Fraco'){
        fazerCaféFraco()
    } else if (intensCafé === 'Médio'){
        fazerCaféMédio()
    } else if (intensCafé === "Forte"){
        fazerCaféForte()
    }
